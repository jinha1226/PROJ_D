# UI/UX & 조작 설계

## 화면 방향
- **세로형(Portrait) 고정**
- 해상도 기준: 1080×2340 (19.5:9) 기준 설계, 다양한 비율 대응
- Safe Area 고려 (노치·펀치홀·하단 홈바)

---

## 화면 레이아웃 구조

```
┌─────────────────────────┐
│  [HP████░░] [MP██░░] B8F │  ← 상단 HUD (높이 48dp)
│  [미니맵]           [🎒] │
├─────────────────────────┤
│                         │
│                         │
│      던전 맵 영역         │  ← 전체의 약 78%
│    (터치 입력 영역)        │
│                         │
│                         │
├─────────────────────────┤
│ [💧][⚡][📖][🧪]│[E1][E2][E3]│[💤] │  ← 하단 HUD (높이 72dp)
└─────────────────────────┘
   퀵슬롯 4개  정수슬롯 3개  휴식
```

## 상단 HUD

### 구성 요소
- **HP 바**: 좌측, 색상 #c03030 → #e05050, 현재/최대값 텍스트
- **MP 바**: HP 바 우측, 색상 #3060c0
- **층수 표시**: 중앙, "B8F" 형식, Cinzel 폰트
- **미니맵**: 좌측 상단 작은 아이콘 — 탭하면 전체 층 지도 화면
- **인벤토리**: 우측 상단 가방 아이콘 — 탭하면 인벤토리 화면

### 주의사항
- 상단 HUD는 최대한 얇게 — 던전 맵 영역 최대화
- HP가 30% 이하 시 HP 바 깜빡임 효과

## 하단 HUD

### 퀵슬롯 (4개)
- 크기: 56×56dp
- 포션, 스크롤, 능력, 주문 자유 등록
- 인벤토리에서 드래그하여 등록
- 탭으로 즉시 발동
- 빈 슬롯은 흐릿한 + 아이콘 표시

### 정수 슬롯 (3개, 이후 4개로 확장 가능)
- 크기: 48×48dp (퀵슬롯보다 살짝 작게)
- 퀵슬롯과 구분선(1px)으로 분리
- 장착된 정수: 계열별 테두리 색상 구분
  - 거인계: #c03030 (빨강)
  - 언데드계: #8a5ac8 (보라)
  - 자연계: #3a9e4a (초록)
  - 원소계: #3a7ec8 (파랑)
  - 심연계: #1a1a2e (검정 테두리 + 글로우)
  - 용계: #e8b84a (금색)
- 빈 슬롯: 흐릿한 ○ 아이콘, "정수 없음" 상태 표시
- 탭하면 정수 교체 팝업 등장

### 휴식 버튼
- 크기: 40×56dp
- 아이콘: 💤 or ZZZ 텍스트
- 탭 → HP 최대까지 자동 대기 (DCSS `Shift+5` 대체)
- 몬스터 시야 진입 시 자동 중단

## 던전 맵 터치 조작

### 기본 조작
| 제스처 | 동작 | 설명 |
|---|---|---|
| 인접 타일 탭 | 이동 | 8방향 모두 지원. 대각선 포함. |
| 적 탭 | 이동 후 공격 | 자동으로 적에게 이동 후 근접 공격 |
| 빈 타일 롱프레스 (0.5초) | 자동탐험 | DCSS `o` 대체. 아이템·몬스터 발견 시 중단 |
| 계단 타일 탭 | 층 이동 확인 팝업 | "B9F로 내려가시겠습니까?" |
| 아이템 타일 탭 | 아이템 팝업 | 줍기 / 사용 / 무시 |
| 문 타일 탭 | 문 열기 | DCSS `o` (open door) 대체 |
| 원거리 공격 대상 탭 | 원거리 공격 | 퀵슬롯에 원거리 무기 등록된 경우 |
| 두 손가락 핀치 | 줌 인/아웃 | 타일 크기 조절 (설정 저장) |

### 자동이동
- 탐험된 타일 탭 → 해당 위치까지 자동이동 (최단경로)
- 이동 중 몬스터 발견 시 자동 중단
- 이동 중 화면 탭 시 즉시 중단

### 조작 불가 상태
- 몬스터 공격 애니메이션 중: 입력 무시 (단, 0.2초 이내)
- 팝업 표시 중: 팝업 외 영역 탭 무시

## 팝업 시스템

### 아이템 팝업
```
┌──────────────────┐
│  🧪 체력 포션      │
│  HP +30 회복      │
│                  │
│ [줍기] [사용] [무시] │
└──────────────────┘
```

### 정수 교체 팝업
```
┌──────────────────────┐
│  슬롯 2 교체          │
│  현재: 본나이트 정수   │
│                      │
│  소지 정수:           │
│  ▸ 뱀 정수 (DEX+4)   │
│  ▸ 트롤 정수 (STR+6) │
│  ▸ [비우기]           │
│                      │
│          [취소]       │
└──────────────────────┘
```

### 레벨업 팝업 (강제 표시)
```
┌──────────────────────┐
│  LEVEL UP! Lv.5      │
│                      │
│  스탯 +1 선택:        │
│  ▸ 힘 (STR) +2       │
│  ▸ 민첩 (DEX) +2     │
│  ▸ 지능 (INT) +2     │
└──────────────────────┘
```
- 레벨업 시 반드시 선택 필요 (DCSS 스탯 선택 계승)
- 선택 전까지 게임 진행 불가

## 캐주얼 / 하드유저 레이어 구조

### 캐주얼 레이어 (항상 표시)
- 타일 터치 이동·공격
- 퀵슬롯 4개
- 정수 슬롯 3개
- 자동탐험 (롱프레스)
- 휴식 버튼
- HP/MP 바

### 하드유저 레이어 (탭하면 접근)
- 인벤토리 전체 화면 (가방 아이콘)
- 캐릭터 상태 전체 (HP 바 탭)
  - 스탯 수치 (STR/DEX/INT/AC/EV)
  - 저항 목록
  - 뮤테이션 목록
  - 현재 신앙 상태
- 전체 층 지도 (미니맵 탭)
- 스킬 수준 확인 (캐릭터 상태 화면 내)

### 설정 화면
- 타일 크기 조절 (핀치 외 슬라이더)
- 애니메이션 속도 (느림/보통/빠름/즉시)
- 자동탐험 중단 조건 설정 (아이템/몬스터/둘다)
- 확인 팝업 on/off (숙련자용)
- 터치 반응 영역 크기 조절

## 세로형 특이사항

- 던전 맵 표시 타일 수: 가로 11~13타일, 세로 18~22타일 (줌 기본값 기준)
- 플레이어는 항상 화면 중앙 고정, 맵이 스크롤
- 줌 아웃 시 최대 가로 17타일까지 (미니맵 대체 가능)
- PC Steam 버전에서도 세로형 유지 (윈도우 크기 고정 or 세로 레이아웃)
  - PC에서는 키보드 방향키·WASD 이동 추가 지원
  - PC에서는 마우스 클릭으로 타일 선택
