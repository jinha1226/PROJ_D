# 직업 & 종족 설계

## 구조
DCSS 계승: 종족 × 직업 조합으로 시작 스탯·특성 결정.
종족이 기본 체형·적성을 결정하고, 직업이 시작 장비·스킬을 결정한다.

---

## 스탯 시스템

### 기본 스탯 (DCSS 계승)
| 스탯 | 약자 | 역할 |
|---|---|---|
| 힘 | STR | 근접 피해, 무거운 갑옷 착용 패널티 감소 |
| 민첩 | DEX | 명중률, 회피(EV), 원거리 피해, 암습 |
| 지능 | INT | 마법 위력·성공률, 마나량 |
| HP | — | 체력. 종족·레벨·STR에 영향 |
| MP | — | 마나. 직업·INT에 영향 |
| AC | — | 갑옷 방어력. 장비 합산 |
| EV | — | 회피. DEX 기반 |

### 레벨업
- 레벨 1마다 HP+3~8 (종족별 상이), MP+2~6
- 레벨 3, 9, 15, 21마다 스탯 +2 선택 (STR/DEX/INT 중 택1)
- 최대 레벨: 27 (DCSS 동일)

## 데이터 구조

```gdscript
# resources/jobs/JobData.gd
class_name JobData
extends Resource

@export var id: String                    # "barbarian"
@export var display_name: String          # "바바리안"
@export var description: String
@export var icon: Texture2D

# 시작 스탯 보너스
@export var str_bonus: int = 0
@export var dex_bonus: int = 0
@export var int_bonus: int = 0

# 시작 장비 ID 목록
@export var starting_equipment: Array[String] = []

# 시작 스킬 레벨
@export var starting_skills: Dictionary = {}   # {"axe": 3, "armour": 2}

# 고유 능력
@export var unique_ability: String = ""        # ability ID

# 정수 상성
@export var essence_affinity: Dictionary = {}  # {EssenceType.GIANT: 0.2}

# 해금 조건
@export var unlock_requirement: String = ""    # meta upgrade ID, "" = 기본해금
```

## 신 시스템 (간소화)

DCSS의 신 시스템을 경량화. 런 중 신전 발견 시 신 선택 가능.
신관 직업은 시작부터 신 선택.

| 신 | 선호 행동 | 보상 능력 | 금지 행동 |
|---|---|---|---|
| 트로그 (전투신) | 근접 전투, 마법 사용 금지 | 분노 지속시간↑, 무기 강화 | 주문 시전 |
| 오쿠 (지식신) | 아이템 감정, 마법 연구 | 마법 위력↑, 자동 감정 | 미감정 아이템 사용 |
| 마크렙 (죽음신) | 언데드 소환, 적 처치 | 시체 소환 강화 | 신성 아이템 사용 |
| 체이브리아도스 (시간신) | 천천히 이동, 대기 | 슬로우 공격 강화, 시간 조작 | 빠른 이동 |

신앙도: 신에게 맞는 행동 시 +1, 금지 행동 시 -5. 0 이하 시 신의 저주.
