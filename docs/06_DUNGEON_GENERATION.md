# 던전 생성 & 구조

## 개요
DCSS의 절차적 던전 생성 방식을 Godot 4로 재구현.
C++ DCSS 소스의 던전 생성 알고리즘을 참고하되 직접 포팅하지 않음.

---

## 던전 전체 구조

```
메인 던전 (D:1 ~ D:15)
├── D:1 ~ D:5   입문 구간 (단순 레이아웃, 약한 몬스터)
├── D:5 브랜치 분기 1
│   ├── 숲 브랜치 (F:1~4)
│   └── 광산 브랜치 (M:1~4)
├── D:6 ~ D:10  중간 구간
├── D:10 브랜치 분기 2
│   ├── 망자의 묘 (G:1~4)
│   └── 마법사 탑 (T:1~5)
├── D:11 ~ D:15 후반 구간
├── D:12 브랜치 분기 3
│   └── 심연 브랜치 (A:1~6) — 클리어 후 개방
└── D:15 최종층 (오브 오브 조트 위치)
```

## 층 생성 알고리즘

### 기본 방식: BSP + 룸 연결
```
1. 맵을 재귀적 BSP(Binary Space Partitioning)로 분할
2. 각 리프 노드에 사각형 방 배치 (랜덤 크기)
3. 인접 방들을 복도로 연결
4. 연결 확인 (모든 방이 도달 가능한지)
5. Vault 배치 (특수 방 삽입)
6. 아이템·몬스터·계단 배치
```

### 맵 크기
- 세로형 화면 기준
- 한 층 맵: 50×80 타일 (가로×세로)
- 화면에 한 번에 표시: 약 13×22 타일 (기본 줌)

### 타일 종류
```gdscript
enum TileType {
    WALL,           # 벽 (이동 불가)
    FLOOR,          # 바닥 (이동 가능)
    DOOR_OPEN,      # 열린 문
    DOOR_CLOSED,    # 닫힌 문
    STAIRS_DOWN,    # 아래 계단
    STAIRS_UP,      # 위 계단
    WATER,          # 물 (수중 이동 가능 종족만 패널티 없음)
    LAVA,           # 용암 (밟으면 피해)
    TRAP,           # 함정 (탐지 전 FLOOR처럼 보임)
    BRANCH_ENTRANCE, # 브랜치 입구
    SHOP,           # 상점
    ALTAR,          # 신전
}
```

## Vault 시스템 (DCSS 참고)

특수 룸 패턴. 층 생성 시 일정 확률로 삽입.

### Vault 데이터 형식 (.vault 파일 또는 GDScript Resource)
```
# resources/vaults/basic_cross.vault
name: "십자형 방"
width: 9
height: 9
weight: 10   # 등장 가중치
min_depth: 1
max_depth: 15

layout:
"#########"
"#...#...#"
"#.......#"
"#...#...#"
"##.###.##"
"#...#...#"
"#.......#"
"#...#...#"
"#########"

# 범례
# # = 벽, . = 바닥, + = 문, > = 계단, ! = 포션 배치점
# @ = 몬스터 배치점, $ = 골드 배치점, ? = 랜덤 아이템
```

## 브랜치별 특성

### 숲 브랜치 (F:1~4)
```
타일: 나무·풀 타일 다수, 물 타일
몬스터: 독사, 거미, 독개구리, 드라이어드, 나무 골렘
특수 기믹: 덩굴 타일 — 이동 시 DEX 체크. 실패 시 1턴 멈춤
보스: 숲의 수호자 (드라이어드 보스, 자연계 소환 능력)
드롭: 자연계 정수, 독 저항 포션, 재생 반지
```

### 광산 브랜치 (M:1~4)
```
타일: 돌 타일, 좁은 복도 많음
몬스터: 오크 전사, 트롤, 광산 골렘, 드워프 감시병
특수 기믹: 붕괴 타일 — 밟으면 일정 확률로 구멍 생성 (아래층으로 이동)
보스: 광산 군주 (거인계 보스, 지진 공격)
드롭: 거인계 정수, 무기 강화석, 금속 방어구
```

### 망자의 묘 (G:1~4)
```
타일: 어두운 돌, 석관 장식물, 어둠 타일 (시야 -2)
몬스터: 좀비, 본나이트, 유령, 미라, 리치
특수 기믹: 어둠 — 기본 시야 -2 (횃불 아이템 중요해짐)
보스: 대리치 (언데드계 보스, 소환 + 저주 공격)
드롭: 언데드계 정수, 저주 해제 스크롤, 부패 방지 부적
```

### 마법사 탑 (T:1~5)
```
타일: 마법진 바닥, 수정 기둥, 마나 분수 (인접 시 MP 회복)
몬스터: 화염정령, 냉기정령, 마법 골렘, 마법사 NPC
특수 기믹: 마나 분수 — 인접 시 MP+5 회복 (1턴 소모)
보스: 대마법사 (원소계 보스, 원소 전환 공격)
드롭: 원소계 정수, 주문서, 마법 완드, 마법 반지
```

### 심연 브랜치 (A:1~6) — 고난이도
```
타일: 완전한 어둠, 공허 타일 (밟으면 순간이동), 용암 다수
몬스터: 공허충, 나락악마, 심연 보스들
특수 기믹: 시야 1타일 고정 (어떤 아이템도 극복 불가)
저주: 입장 시 랜덤 저주 1개 적용
보스: 심연의 군주 (보스 중 최강, 심연계+용계 정수 드롭)
```

## 몬스터 배치 규칙

### 깊이별 몬스터 강도
```gdscript
func get_monsters_for_depth(depth: int) -> Array:
    # depth: 1~15 (메인), 브랜치는 별도 테이블
    var tier = ceili(depth / 3.0)  # 1~5 티어
    return MONSTER_TABLE[tier]
```

| 티어 | 층수 | 등장 몬스터 |

## 아이템 배치

### 층당 아이템 수
- D:1~5: 아이템 4~7개
- D:6~10: 아이템 5~8개
- D:11~15: 아이템 6~10개

### 아이템 종류 (DCSS 계승)
```
무기 (Weapon): 근접·원거리 무기
갑옷 (Armour): 몸통·방패·투구 등
장신구 (Jewellery): 반지·부적 — 착용 시 효과
포션 (Potion): HP 회복, 마나 회복, 강화, 변이 등
스크롤 (Scroll): 감정, 강화, 저주해제, 텔레포트 등
완드 (Wand): 사용 가능 횟수 제한 마법 도구
음식 (Food): 허기 회복 (DCSS 계승, 간소화)
```

## 계단 & 층 이동

### 계단 규칙
- 각 층에 내려가는 계단 2~3개, 올라가는 계단 1~2개
- 처음 방문하는 층: 계단 위치 미공개 (탐험 필요)
- 메타 업그레이드 "탐험가의 직감": 내려가는 계단 1개 위치 공개

### 브랜치 입구
- 브랜치 입구는 특수 타일로 표시 (일반 계단과 구분)
- 입구 발견 시: "숲 브랜치 입구를 발견했습니다. [진입] [무시]" 팝업
- 브랜치 진입 후 돌아오기: 브랜치 첫 층에 항상 메인 던전 귀환 계단 존재

const MAX_ROOM_SIZE = 12
